# WhatsApp-Chat-Reader

![alt text](grafik.png)

1. Chat per E-Mail versenden
2. Zip-Datei entpacken und in das gleiche Verzeichnis legen
3. $file, $leftname, $rightname in read.php bearbeiten
4. Webseite aufrufen
